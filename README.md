## иҝҷжҳҜд»Җд№Ҳпјҹ

FinAgent жҳҜдёҖдёӘејҖжәҗзҡ„ A иӮЎжҷәиғҪжҠ•з ”еҠ©зҗҶпјҢеҹәдәҺ [Nanobot](https://github.com/HKUDS/nanobot) иҪ»йҮҸзә§ AI Agent жЎҶжһ¶жһ„е»әпјҢйҖҡиҝҮйЈһд№ҰжңәеҷЁдәәиҝӣиЎҢдәӨдә’гҖӮ

**е®ғдёҚжҳҜеҸҰдёҖдёӘ"AIйҖүиӮЎе·Ҙе…·"гҖӮ** еёӮйқўдёҠеӨ§еӨҡж•° AI зӮ’иӮЎе·Ҙе…·зҡ„еҒҡжі•жҳҜзҪ—еҲ—жҢҮж ҮгҖҒе Ҷз Ңж•°жҚ®пјҢз„¶еҗҺз»ҷдҪ дёҖдёӘжЁЎзіҠзҡ„"жҺЁиҚҗд№°е…Ҙ"гҖӮFinAgent дёҚеҗҢвҖ”вҖ”е®ғзҡ„ж ёеҝғи®ҫи®ЎеҺҹеҲҷжҳҜ**йҖ»иҫ‘й—ӯзҺҜ**пјҡжҜҸдёҖжқЎеҲҶжһҗз»“и®әйғҪеҝ…йЎ»жңүе®Ңж•ҙзҡ„еӣ жһңжҺЁеҜјй“ҫжқЎпјҢе№¶дё”жҳҺзЎ®з»ҷеҮә"д»Җд№Ҳжғ…еҶөдёӢйҖ»иҫ‘дёҚжҲҗз«Ӣ"гҖӮ

```
рҹ“° и§ҰеҸ‘дҝЎеҸ·
  вҶ’ рҹ§  з¬¬дёҖжҖ§еҺҹзҗҶжҺЁеҜј
    вҶ’ рҹ”— дә§дёҡй“ҫжҳ е°„
      вҶ’ рҹ“Ҡ ж•°жҚ®йӘҢиҜҒ
        вҶ’ рҹ’° дј°еҖјдёҺдҪҚзҪ®
          вҶ’ вң… ж“ҚдҪңе»әи®® + вқҢ еӨұж•ҲжқЎд»¶
```

## ж ёеҝғеҠҹиғҪ

### рҹ“Ҡ йҖ»иҫ‘еҲҶжһҗ

иҫ“е…ҘдёҖеҸӘиӮЎзҘЁд»Јз ҒпјҢиҺ·еҫ—дёҖд»Ҫ**йҖ»иҫ‘й—ӯзҺҜйӘҢиҜҒжҠҘе‘Ҡ**пјҢиҖҢдёҚжҳҜдј з»ҹз ”жҠҘгҖӮ

- **е•ҶдёҡжЁЎејҸиҜҶеҲ«** вҖ” иҝҷе®¶е…¬еҸёиөҡй’ұзҡ„жң¬иҙЁжҳҜд»Җд№Ҳпјҹ
- **жҠӨеҹҺжІіиҜ„дј°** вҖ” еЈҒеһ’жҳҜеңЁеҠ ејәиҝҳжҳҜеүҠејұпјҹ
- **ж”ҝзӯ–жҳ е°„** вҖ” жңүж— еӣҪе®¶зә§ж”ҝзӯ–ж”Ҝж’‘пјҹ
- **иҙўеҠЎйӘҢиҜҒ** вҖ” ж•°жҚ®жҳҜеҗҰж”Ҝж’‘йҖ»иҫ‘пјҹ
- **й—ӯзҺҜз»“и®ә** вҖ” иөҡй’ұйҖ»иҫ‘жҳҜеҗҰжҲҗз«Ӣ + еӨұж•ҲжқЎд»¶

### рҹ“Ў еёӮеңәйӣ·иҫҫ

дёҚжҳҜз®ҖеҚ•зҡ„ж–°й—»иҒҡеҗҲпјҢиҖҢжҳҜ**д»Һж–°й—»/ж”ҝзӯ–дёӯжҺЁеҜјжҠ•иө„йҖ»иҫ‘й“ҫ**гҖӮ

- **еӨ§зӣҳдҪ“жё©и®Ў** вҖ” жғ…з»ӘжҢҮж ҮпјҲжһҒеәҰд№җи§Ӯ/и§Ӯжңӣ/жҒҗж…Ңпјү
- **йҖ»иҫ‘й“ҫжҺЁеҜј** вҖ” дәӢд»¶вҶ’дә§дёҡеҪұе“ҚвҶ’еҸ—зӣҠзҺҜиҠӮвҶ’е…·дҪ“ж Үзҡ„
- **дёүжўҜйҳҹжҳ е°„** вҖ” зӣҙжҺҘеҸ—зӣҠ / й—ҙжҺҘеҸ—зӣҠ / дё»йўҳеҸ—зӣҠ
- **жҢҒд»“е…іиҒ”** вҖ” иҮӘеҠЁдәӨеҸүйӘҢиҜҒзғӯзӮ№дёҺдҪ зҡ„жҢҒд»“
- **жҜҸж—ҘжҺЁйҖҒ** вҖ” зӣҳеүҚ08:30 / зӣҳеҗҺ15:15 иҮӘеҠЁжҺЁйҖҒйЈһд№Ұ

### рҹ“Ҳ KзәҝеҶізӯ–

дёҚжҳҜеұ•зӨәжҢҮж Үж•°еҖјпјҢиҖҢжҳҜ**зӣҙжҺҘеӣһзӯ”"зҺ°еңЁиҜҘжҖҺд№Ҳж“ҚдҪң"**гҖӮ

- **йҮҸиғҪеҪўжҖҒ** вҖ” еҖҚйҮҸ / зј©еҖҚйҮҸ / жўҜйҮҸ / ең°йҮҸиҜҶеҲ«
- **и¶ӢеҠҝеҲӨж–ӯ** вҖ” еқҮзәҝжҺ’еҲ— + еӨҡе‘Ёжңҹе…ұжҢҜ
- **еҶізӯ–иҫ“еҮә** вҖ” жҢҒжңү/еҠ д»“/еҮҸд»“/жё…д»“ + жӯўзӣҲжӯўжҚҹдҪҚ + зӣҲдәҸжҜ”
- **йҖ»иҫ‘еӣһжЈҖ** вҖ” еҪ“еҲқд№°е…Ҙзҡ„йҖ»иҫ‘жҳҜеҗҰиҝҳеңЁпјҹ

### вҡЎ жү§иЎҢдёҺйЈҺжҺ§

- **йЈһд№ҰдәӨдә’** вҖ” иҮӘз„¶иҜӯиЁҖжҢҮд»ӨпјҲ`жҹҘ600519` / `зңӢеӨ§зӣҳ` / `Kзәҝ 300750`пјү
- **жқЎд»¶йў„иӯҰ** вҖ” д»·ж јзӘҒз ҙ/и·Ңз ҙгҖҒеҖҚйҮҸејӮеҠЁгҖҒMACDйҮ‘еҸүзӯү
- **дәӨжҳ“йў„з•ҷ** вҖ” QMT/PTrade/йҖҡиҫҫдҝЎжҺҘеҸЈйў„з•ҷпјҢHITLдәәе·ҘзЎ®и®Ө
- **йЈҺжҺ§зЎ¬йҷҗ** вҖ” еҚ•иӮЎжңҖеӨ§д»“дҪҚгҖҒеҚ•ж—ҘдәҸжҚҹдёҠйҷҗгҖҒжҖ»еӣһж’ӨдҝқжҠӨ

### дёүеұӮдёҡеҠЎжһ¶жһ„

| еұӮзә§ | иҒҢиҙЈ | е…ій”®з»„д»¶ |
|------|------|---------|
| **L3 дәӨдә’жү§иЎҢеұӮ** | з”ЁжҲ·дәӨдә’ + дәӨжҳ“жү§иЎҢ | йЈһд№ҰBot В· Cronе®ҡж—¶ В· дәӨжҳ“зҪ‘е…і(QMT/PTrade/TDX) |
| **L2 жҷәиғҪжҺЁзҗҶеұӮ** | AIжҺЁзҗҶ + жҢҮж Үи®Ўз®— | Nanobot Agent Loop В· LLM В· Skills В· pandas_ta |

### ж•°жҚ®йҖӮй…ҚеҷЁ

жүҖжңүиЎҢжғ…ж•°жҚ®жәҗе®һзҺ°еҗҢдёҖдёӘжҠҪиұЎжҺҘеҸЈпјҢж–°еўһеёӮеңәеҸӘйңҖе®һзҺ°еӯҗзұ»пјҡ

```python
class DataAdapter(ABC):
    """з»ҹдёҖж•°жҚ®йҖӮй…ҚеҷЁжҺҘеҸЈ"""
    def get_kline(self, symbol, period, count) -> pd.DataFrame: ...
    def get_realtime_quote(self, symbol) -> dict: ...
    def get_financials(self, symbol) -> dict: ...
    def get_fund_flow(self, symbol) -> dict: ...
    def get_news(self, keywords, days) -> list: ...

class TradeAdapter(ABC):
    """з»ҹдёҖдәӨжҳ“жҺҘеҸЈ"""
    def place_order(self, symbol, side, qty, price, order_type): ...
    def cancel_order(self, order_id) -> bool: ...
    def get_positions(self) -> list: ...
    def get_account_info(self) -> dict: ...

# дёҖжңҹе®һзҺ°
class QMTAdapter(DataAdapter): ...      # иҝ…жҠ•QMT (xtquant)
class AKShareAdapter(DataAdapter): ...  # AKShare (е…Қиҙ№)

# йў„з•ҷ
class PTradeAdapter(DataAdapter): ...   # жҒ’з”ҹPTrade
class TDXAdapter(DataAdapter): ...      # йҖҡиҫҫдҝЎ
class USStockAdapter(DataAdapter): ...  # зҫҺиӮЎ (yfinance / IB)
class CryptoAdapter(DataAdapter): ...   # еҠ еҜҶиҙ§еёҒ (CCXT)
```

## и®ҫи®Ўе“ІеӯҰ

### йҖ»иҫ‘й—ӯзҺҜзҡ„дә”жқЎй“ҒеҫӢ

1. **еҸҜиҝҪжәҜ** вҖ” з”ЁжҲ·й—®"дёәд»Җд№ҲжҺЁиҚҗиҝҷдёӘ"пјҢзі»з»ҹиғҪз»ҷеҮә 3 еұӮд»ҘеҶ…жҺЁзҗҶ
2. **еҸҜиҜҒдјӘ** вҖ” жҜҸдёӘз»“и®әеҝ…йЎ»йҷ„еёҰ"йҖ»иҫ‘еӨұж•ҲжқЎд»¶"
3. **дёҚй»‘зӣ’** вҖ” дёҚдҫқиө–дёҚеҸҜи§ЈйҮҠзҡ„жЁЎеһӢпјҢдјҳе…Ҳз”Ёе…¬ејҖеҸҜзҗҶи§Јзҡ„йҖ»иҫ‘
4. **дёӯзәҝе®ҡдҪҚ** вҖ” ж—Ҙзәҝдёәдё»пјҢе‘ЁзәҝзЎ®и®ӨпјҢдёҚеҒҡж—ҘеҶ…еҷӘйҹі
5. **йЈҺйҷ©дјҳе…Ҳ** вҖ” е…Ҳй—®"дјҡдәҸеӨҡе°‘"пјҢеҶҚй—®"иғҪиөҡеӨҡе°‘"
